Page Setup command (File menu)

The "Page Setup" menu choice allows you to set the printed report length, the top and left margins, and the point size. The dialog box has the following entries:

  • Top margin: Top margin for a printed report.
  • Left margin: Left margin for a printed report.
  • Font size: Font size in points for a printed report.

After you have modified the report length, margins, or font size, the new values are saved in the Registry. Press Cancel to exit from the dialog box without saving any new values to the Registry.
