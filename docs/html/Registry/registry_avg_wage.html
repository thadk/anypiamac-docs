Average Wage Increases Set x section in the Registry

There can be up to four sections in the Registry with titles of Average Wage Increases Set x, where x is 1, 2, 3, or 4. Each section should have 13 lines. Each line has two parts (a Name entry and a Data entry):

Data entry
