Input data from a file

Input can be entered from a file by choosing File from the main menu and Open... from the pulldown menu. If the file was originally created by this program, then it will automatically be in the correct format to be read back into the program. It is possible to create a file with the required information, in the correct format, outside of this program (for instance, in another program running against a database of worker information). However, the file created must be in exactly the correct format or this program will detect an error and stop. See File structure for a description of the required file structure.
