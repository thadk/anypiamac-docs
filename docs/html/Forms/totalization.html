Totalization benefit (Worker Info form)

Check the radio button if you are requesting a totalization benefit. However, be aware that even if you are not eligible for totalization benefits, the program will calculate a hypothetical benefit which is generally greater than the non-totalization benefit which would actually be payable. Therefore, do not select the totalization button if you are not eligible for totalization.

If you check this button, you must enter annual earnings and quarters of coverage, beginning with the first year of earnings (even if before 1950). The figures must be accurate to produce an accurate totalization benefit.
